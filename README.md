# vue-on-docker

## Install
1. Install docker, docker-compose

## How to build
1. Enter the project
   ```cd <project directory>```
2. Build docker
   ```docker-compose up -d```
3. Enter the container
   ```docker-compose exec web sh```
4. Build new vue app
   ```vue create <project-name> -d```
   Build Nuxs.js app
   ```yarn create nuxt-app <project-name>```
